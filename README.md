# seriousgame_bericht

Praktikumsbericht (ca. 20 Seiten) zur Entwicklung einer Serious-Game-Plattform im Gesundheitskontext.

## Inhalte
- Einleitung: Thema, Relevanz und bestehende Lösungen
- Zielübersicht des Praktikums
- Monatliche Tätigkeitsbeschreibung (mehrmonatiger Verlauf)
- Fachliche/technische Ergebnisse
- Herausforderungen und Lessons Learned
- Fazit und Ausblick
- Platzhalter für Screenshots
- Anhang (README-Auszug)

## Overleaf-kompatibel
Dieses Repository ist als Overleaf-Projekt vorbereitet.

- Hauptdatei: `main.tex`
- LaTeX-Engine: pdfLaTeX (Standard)

## Lokales Bauen
```bash
pdflatex main.tex
```
